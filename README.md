# nikitayo
Данная программа представляет собой консольное приложение на языке C++, реализующее демонстрационный алгоритм шифрования — шифр Цезаря.
Программа запрашивает у пользователя ввод строки (пароля), после чего выполняет её шифрование с использованием фиксированного ключа сдвига, равного 3. Затем зашифрованная строка выводится на экран, после чего выполняется обратная операция — расшифровка — для проверки корректности работы алгоритма.
Шифрование реализовано путём смещения числового кода каждого символа на заданное значение с использованием операции взятия по модулю 256. Это позволяет корректно обрабатывать весь диапазон символов ASCII и предотвращает переполнение.
Основные функции программы:
caesarEncrypt() — выполняет посимвольное шифрование строки.
caesarDecrypt() — выполняет обратное преобразование (расшифровку).
main() — отвечает за ввод данных, вызов функций и вывод результата.
Программа носит учебный характер и предназначена для демонстрации принципов симметричного шифрования. Используемый алгоритм не является криптографически стойким и не предназначен для реального хранения паролей.
